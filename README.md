# Portfolio — Jose Eliuth Torres Ortiz

Personal portfolio website for a **Junior Data Analyst & Data Scientist**. Built with Next.js and Tailwind CSS, bilingual (ES/EN), and deployed on Vercel.

**Live demo:** [eliuth-torres.vercel.app](https://eliuth-torres.vercel.app)

---

## Tech Stack

| Layer      | Technology                  |
|------------|-----------------------------|
| Framework  | Next.js 16 (App Router)     |
| Styling    | Tailwind CSS                |
| Language   | TypeScript                  |
| Deployment | Vercel                      |

---

## Sections

| Section    | Description                                              |
|------------|----------------------------------------------------------|
| Hero       | Name, title, short bio, and call-to-action buttons       |
| About      | Background, education summary, and key stats             |
| Skills     | Visual grid of technical skills (Python, SQL, ML, etc.)  |
| Projects   | Cards for Predict-Heart, Spotify ETL, and EvoNeuroGraph  |
| Experience | Timeline entry for SSPT (Sep 2021 – Apr 2024)            |
| CV         | Download button for the PDF resume                       |
| Contact    | Links to GitHub, LinkedIn, and email                     |

---

## Run Locally

```bash
git clone https://github.com/inter097/portfolio.git
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

> **Note:** Replace `public/cv.pdf` with your actual PDF resume before deploying.

---

## Deploy on Vercel

1. Push the repository to GitHub.
2. Go to [vercel.com](https://vercel.com) → **Add New Project** → import the repo.
3. No extra configuration needed — Vercel detects Next.js automatically.
4. Every push to `main` triggers a new deployment.
